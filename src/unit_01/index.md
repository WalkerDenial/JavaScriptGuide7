# JavaScript 简介

JavaScript 是 Web 编程语言。是一门 **高级**、**动态**、**解释型** 编译语言，非常适合面向对象和函数式编程风格。

JavaScript  的变量是无类型的，它的语法大致与 Java 相仿，但除此之外这两门语言之间没有任何关系。

JavaScript 从 Scheme 借鉴了一类（first class）函数，从不太知名的 Self 借鉴了基于原型的继承。
